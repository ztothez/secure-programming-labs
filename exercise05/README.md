# Exercise 05 – Safe CLI Calculator (C++)

This exercise implements a command-line calculator with strict input validation.

File included:

- `safe_calculator.cpp`

Security features:

- Safe numeric parsing using `strtold`
- Rejection of NaN and infinity
- Operator validation
- Division-by-zero checks
- Finite result verification
- Clear exit codes for different error cases

The goal is to demonstrate defensive programming in argument parsing and arithmetic operations.
